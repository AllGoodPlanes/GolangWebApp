GolangWebApp is in active development

It is a blank app template written in GOlang, an open source programming language that makes it easy to build simple, reliable and efficient software.

Visit the GO site for more information

The purpose of developing this simple site is to learn all aspects of Golang web development using only the standard GO library, i.e. without using 3rd party developed packages.
